## 재귀함수에서의 실수 2022년 2월 27일

```python
# 재귀함수에서 실수 했음. 초기화 조건 깜빡!!!!!! 꼭 확인할 것!
def fact(n):
	if n = 0 or n = 1: return 1  # 이런 초기화 조건이 재귀함수에서 반드시 필요함
	return n * (n - 1)

```
